Hotel Management System

A full-stack Hotel Management System built using Spring Boot that helps manage hotel operations such as room booking, customer management, reservations, check-in/check-out, billing, and room availability.

📌 Features
User Registration & Login Authentication
Admin Dashboard
Room Management
Room Booking & Reservation
Check-In / Check-Out System
Customer Management
Billing & Payment Handling
Room Availability Tracking
Booking History
REST API Integration
Responsive User Interface
